# SmartLight<sup> by Samokhin Aliaksei</sup>
Проект "SmartLight" позволяет управлять адресной светодиодной линией (WS2812, WS2811) и создавать различные световые эффекты на потолочных световых линиях. Уникальность проекта в том, что в контроллер (с помощью веб-сервера) отправляются данные о типе и геометрических размерах световой линии. Тем самым анимация может возпроизводиться как в прямом так и в обратном направлении, т.к. контроллер знает количество физических пикселей и длину светодиодной ленты.

### Файлы
Весь проект поделен на папки для удобства навигации и отладки. Ниже описание файлов и краткое описание функций в этих файлах.
* ./ ***index.html*** - основной html файл веб-сервера.

_**./COMPONENTS**_
* ./components/anim1 ***anim1.css*** - все анимации для для рисунка DRAW-01 (Прямоугольник)
* ./components/anim1 ***anim1.svg*** - геометрия рисунка DRAW-01 (Прямоугольник) для окна Предпросмотр на вкладке "Установки".

_**./CONFIG**_
* ./config/ ***settings.json*** - основной файл настроек между контроллером и веб-сервером. Файл загружается в момент запуска веб-сервера. При его отсутствии на сервере крутится Progress indicator и в консоли браузера будет ошибка.
* ./config/ ***draws.json*** - файл содержащий описание всех возможных рисунков.
	* *id* - id рисунка для определения его в JS.
	* *label* - имя рисунка в модальном окне для пользователя.
	* *svg* - эскиз рисунка для модального окна в формате *.svg.
	* *perspective* - ссылка на *.svg для анимаций в перспективном виде.
	* *extra-dims* - дополнительные размеры для описания геометрии рисунка, если они требуются.
	* *effects* - карта доступных эффектов для конкретного рисунка.
* ./config/ ***effect-map.json*** - карта эффектов которые можно применить к данному рисунку. 

_**./CSS**_
* ./css/ ***m3-theme.css*** - основной файл стилей оформления веб-сервера.

_**./JS**_
* ./js/ ***app.js*** - основной JS файл.
* ./js/modules/ ***dims.js*** - файл для работы с *полями* и *дополнительными полями* размеров.
* ./js/modules/ ***draw.js*** - логика эскизов и модального окна выбора.
* ./js/modules/ ***effects.js*** - логика эффектов анимации.
* ./js/modules/ ***loader.js*** - загрузка конфигов и данных.
* ./js/modules/ ***power.js*** - файл создан для расчета энергопотребления ленты. На вкладке "Главная" есть ползунок потребления. Логика пересчета длины всей ленты в этом файле.
* ./js/modules/ ***state.js*** - все DOM-переменные и состояние приложения.
* ./js/modules/ ***ui.js*** - обновление интерфейса (питание, навигация, wifi).

### ID рисунков

* draw-02 - угол с размерами: Длина и Ширина
* draw-03 - крест с размерами: Длина, Ширина и экстра размеры A и B
* draw-04 - квадрат в квадрате: Длина, Ширина и экстра размеры A и B

* anim-06 - волна яркости

### Поставка
Контроллер поставляется в корпусе.

### Подключение
Контроллер питается постоянным напряжением 5-12В. Питание необходимо подавать на соответствующие разъемы соблюдая полярность, указанную на корпусе контроллера. Контроллер имеет защиту от переполюсовки.
