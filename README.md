# SmartLight
Проект "SmartLight" позволяет управлять адресной светодиодной линией (WS2812, WS2811) и создавать различные световые эффекты на потолочных световых линиях. Уникальность проекта в том, что в контроллер (с помощью веб-сервера) отправляются данные о типе и геометрических размерах световой линии. Тем самым анимация может возпроизводиться как в прямом так и в обратном направлении, т.к. контроллер знает количество физических пикселей и длину светодиодной ленты.

### Файлы
Здесь будет описание функций и для чего они созданы.
- ./config/**settings.json** - основной файл настроек между контроллером и веб-сервером. Файл загружается в момент запуска веб-сервера.

- ./js/modules/**dims.js** - файл для работы с *полями* и *дополнительными полями* размеров.
- ./js/modules/**power.js** - файл создан для расчета энергопотребления ленты. На вкладке "Главная" есть ползунок потребления. Логика пересчета длины всей ленты в этом файле.

### Поставка
Контроллер поставляется в корпусе.

### Подключение
Контроллер питается постоянным напряжением 5-12В. Питание необходимо подавать на соответствующие разъемы соблюдая полярность, указанную на корпусе контроллера. Контроллер имеет защиту от переполюсовки.

### Технические характеристики
- Питание: 5-12В DC
- Потребление: 200мА
- Потребление в режиме сна: 0мА
- Время запуска: 20мс с момента подачи питания на контроллер
